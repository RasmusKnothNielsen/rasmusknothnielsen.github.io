Tools

NPM

npm is used to install packages from a large software registry.

                $ npm install [package]
                $ npm i [package]
                $ npm install -g [package]

the -g flag means that the package is installed globally, thus it can be accessed from all projects.

NPX

npx is used to run a package from npm.

                $ npx budo index.js --live --open

This runs the budo package from npm, with two flags. Check Budo for more information.

Budo

Enables live reloading of server when changes occur. Does what nodemon does,
but with the added benefit of auto refreshing the browser.

                $ npx budo index.js --live --open

Using the npx budo commmand to run the index.js file with live reload flag and the open flag. Opens up the browser

Nodemon

Enables automatic restart of server, when changes in the directory is deteched

                $ nodemon app.js

Manipulation of filesystem

How to check what is in a dir. (ls)

                fs.readdir(__dirname + '/public/', function(err, items) {

                    for (let i = 0; i < items.length; i++) {
